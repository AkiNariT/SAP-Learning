# 定义 G/L 勘定グループ

T-cd: OBD4

勘定グループ控制两个东西：<br>
1. 这个 G/L 科目号允许在哪个番号范围内
2. 创建 G/L 主数据时，哪些字段出现/必输/可选

例如：

```text
100000～399999 = 资产负债类科目
400000～899999 = 损益类科目
900000～999999 = 结转/特殊科目
```

| 勘定グループ | 推荐名称      | 中文理解                | 号码范围            |
| ------ | --------- | ------------------- | --------------- |
| `ZBS`  | `貸借対照表勘定` | 资产负债表科目，例如应收、库存、现金  | `100000～399999` |
| `ZPL`  | `損益計算書勘定` | 损益科目，例如销售收入、销售成本、费用 | `400000～899999` |
| `ZSP`  | `特殊・繰越勘定` | 特殊科目，例如损益结转科目       | `900000～999999` |

简单理解：<br>
ZBS：资产负债类<br>
ZPL：收入成本费用类<br>
ZSP：特殊结转类<br>
