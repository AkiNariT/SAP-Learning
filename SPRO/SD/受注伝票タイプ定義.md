# 受注伝票タイプ定義

T-CD： VOV8 <br>
暂时选择自建。用来练习。可以完全使用`OR`。

| 字段       | 建议值 / 标准值 | 说明          |
| -------- | --------- | ----------- |
| 販売伝票タイプ  | `LDOR`      | 标准销售订单      |
| 販売伝票カテゴリ | `C`       | 受注          |
| 番号範囲内部割当 | 有值即可      | 销售订单内部编号    |
| 明細番号増分   | `10`      | 明细 10、20、30 |
| 伝票価格決定表  | 通常 `A`    | 后面价格过程决定用   |
| 出荷タイプ    | `LF`      | 默认出荷类型      |
| 請求タイプ    | `F2`      | 默认請求类型      |

如果 VA01 后面报号码范围错误，要维护销售单据号码范围。<br>
使用`VN01`。<br>
