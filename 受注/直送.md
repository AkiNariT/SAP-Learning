# 直送
# 標準受注 と 直送 の比較

| 項目 | 標準受注 | 直送 |
|---|---|---|
| 業務流れ | 受注 → 出荷 → 出庫転記(PGI) → 請求 | 受注 → 購買依頼自動生成 → 購買発注 → 仕入先が得意先へ直接出荷 → 仕入先請求 → 得意先請求 |
| 受注 | VA01 | VA01 |
| 出荷 | VL01N / VL02N | なし（自社出荷なし） |
| 出庫転記 | PGI | なし |
| 購買依頼 | なし | 自動生成 |
| 購買発注 | なし | ME21N |
| 仕入先請求 | なし | MIRO |
| 得意先請求 | VF01 | VF01 |
| 在庫経由 | 自社在庫を経由する | 自社在庫を経由しない |
| 主な連携 | SD中心 | SD + MM 連携 |

## 流程イメージ

### 標準受注
- 受注（VA01）
- ↓
- 出荷（VL01N / VL02N）
- ↓
- 出庫転記（PGI）
- ↓
- 請求（VF01）

### 直送
- 受注（VA01）
- ↓
- 購買依頼（自動生成）
- ↓
- 購買発注（ME21N）
- ↓
- 仕入先が得意先へ直接出荷
- ↓
- 仕入先請求（MIRO）
- ↓
- 得意先請求（VF01）
