<h2>动态的使用</h2>

<h3>最小动态内表例子：动态表名</h3>

   ```abap  
PARAMETERS p_tab TYPE tabname DEFAULT 'VBAK'.

   ```
想运行时决定查哪张表：<dr/>
   ```abap  
FIELD-SYMBOLS <gt_data> TYPE STANDARD TABLE.

DATA lr_data TYPE REF TO data.

CREATE DATA lr_data TYPE STANDARD TABLE OF (p_tab).
ASSIGN lr_data->* TO <gt_data>.

SELECT *
  FROM (p_tab)
  INTO TABLE @<gt_data>
  UP TO 100 ROWS.
   ```
