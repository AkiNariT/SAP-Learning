## 简单理解
BTP 是“开发、扩展、集成平台”。<br>
BDC 是“企业数据、分析、AI 的数据平台”。<br>

所以BDC更贴近数据分析！要把数据变成人话，只不过数据分析变成了AI。<br>

对SAP冲击的话则是BW模块。因为BW本身就是在分析数据。<br>




<br>
管 SAP/非 SAP 数据，做数据产品、分析、AI 数据基础<br>
BDC：SAP Business Data Cloud<br>
而 BDC 官方定位更偏向 统一和治理 SAP 与非 SAP 数据，支撑高级分析和 AI。<br>
<br>

你做 BDC 项目时，可能做：
- SAP Datasphere 建模
- SAP BW 数据迁移/现代化
- SAC 报表分析
- 数据产品设计
- SAP Databricks 数据处理
- 给 AI/分析提供统一数据基础
